# 文件名称 module
本文件负责管理EZEngine的程序资产库，本文件将模块的各个责任分散到不同的文件夹中管理

程序资产目前被分为

- 数据协议  ：仅包含数据定义，强制要求所有数据类的序列化和反序列化，需要是可以存储的数据

- 基础协议  : 负责抽象基础对象，如 thead , Entity 等。通常是运行时数据,可以作为component存在，通过引用数据协议实现序列化

- 控制器协议：对内维护一个自己模块的上下文，对外提供访问和修改这个上下文的方法

- 系统      ：通过引用以上协议，实现对自己模块的数据更新

- 启动器    ：包装了一个程序完整周期的资产，持有一个可扩展的世界上下文，作为脚本访问世界的入口

- 工具      ：通常是编辑器时期的辅助脚本，未来将承载gizmo等功能



## 功能特性
- 作为程序资产文件夹存在

## 环境依赖
- EZEngine/src/  引擎组装程序的位置
